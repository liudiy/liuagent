## 第 8 页

2. 如何选择产品
注: 除提供给用户相应服务器版本外，再提供 TongWeb Windows 版本以供用户测试使用。
类别
选择产品
• Spring Boot3.x
• Spring Framework6.x 框
架需要 Jakarta EE 9 规范
• war 应用：选择 TongWeb8.0/7.0.8
• java -jar 运行方式：选择 TongWeb7.0.E/8.0.E

TongWeb7.0.4.X 版本不支持 Spring Boot3.x 、Spring Framework6.x 
框架。
tomcat10 上的应用
选择 TongWeb8.0/7.0.8

TongWeb7.0.4.X 版本不支持 tomcat10 上应用。
Spring Cloud Gateway
仅 TongWeb7.0.E/8.0.E 嵌入版支持
云平台
根据应用的使用方式选择 “企业版”、“容器云版” 或 “嵌入版”
专用机
• TongWeb7.0 版本：根据 CPU 型号、操作系统版本号、服务器/客户
机、deb/rpm 包，发对应版本
• TongWeb8/7.0.8 版本：可在专用机上直接解压使用，若需要安全管
理器正规方式安装，再做 deb/rpm 包
